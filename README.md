# Article CMS Flask Application on Azure

## Project Overview

This project is a Flask-based Article Content Management System (CMS) deployed using Microsoft Azure services. The application allows users to log in, create and manage articles, and upload images. The project demonstrates cloud deployment, database integration, blob storage usage, and web application hosting on Azure.

---

## Features

- User Login Authentication
- Create and Edit Articles
- Image Upload Support
- Azure SQL Database Integration
- Azure Blob Storage Integration
- Flask Web Framework
- Azure App Service Deployment

---

## Azure Services Used

- Azure App Service
- Azure SQL Database
- Azure SQL Server
- Azure Blob Storage
- Azure Resource Group

---

## Technologies Used

- Python
- Flask
- SQLAlchemy
- Flask-Login
- Flask-WTF
- HTML/CSS
- Azure Services
